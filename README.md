
# MailHog with Laravel
```
docker-compose up -d
```
# Access panel

Access http://localhost:8100, to use the Mailhog Panel

# Change variables in your Laravel .env

```
MAIL_MAILER=smtp
MAIL_HOST=localhost
MAIL_PORT=8003
MAIL_USERNAME=''
MAIL_PASSWORD=''
MAIL_ENCRYPTION=null
```
